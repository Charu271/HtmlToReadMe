Html To Read Me


Love to work on GitHub? But wasting time on creating Mark-ups for ReadMe?



How It Works

We all stuck at ReadMe while creating a Repo on GitHub. Here you can use Html code that will be converted into Markup for GitHub to help you save your time and create cool ReadMe


Just Type in your Html Code into the given input box, click on convert and TADA! you are ready to use your ReadMe

Example Code

Html Code

<h1>My Sample Project</h1>
<ul> <li>My first list Item<h1> <ul>

ReadMe Code

# My Sample Project
* My first list Item
